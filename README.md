# 墨影格斗（单文件网页版）

这是一个**只有 `index.html` 一个网页文件**的黑白水墨风 1v1 格斗小游戏。

## 特色
- 4 个可选角色（速度、伤害、攻击距离不同）
- 快速按键响应：移动、跳跃、防御、轻击、重击、冲刺
- 打击反馈：命中停顿（Hit Stop）、镜头震动、墨迹粒子
- 黑白墨色视觉风格（参考暗影格斗）

## 操作
- `A / D`：左右移动
- `W`：跳跃
- `S`：防御（减伤）
- `J`：轻击
- `K`：重击
- `L`：冲刺

## 运行
直接双击 `index.html` 即可。

或：
```bash
python -m http.server 8000
```
访问 <http://localhost:8000>
