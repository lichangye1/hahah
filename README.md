# 墨影格斗（网页版）

一个黑白水墨风的 1v1 格斗小游戏，强调输入响应和打击反馈，风格参考暗影格斗。

## 特色
- 4 个可选角色（速度、伤害、攻击距离各不相同）
- 快速按键响应：轻击、重击、防御、冲刺、跳跃
- 打击感：命中停顿（Hit Stop）、镜头震动、墨迹粒子喷溅
- 黑白墨色美术风格

## 操作
- `A / D`：左右移动
- `W`：跳跃
- `S`：防御（减伤）
- `J`：轻击
- `K`：重击
- `L`：冲刺

## 运行
直接双击 `index.html`，或在目录启动静态服务器：

```bash
python -m http.server 8000
```

然后访问 <http://localhost:8000>
